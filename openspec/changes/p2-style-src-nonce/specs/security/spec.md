## MODIFIED Requirements

### Requirement: style-src Compatibility
The CSP style-src directive SHALL be compatible with Next.js inline style handling.

#### Scenario: Inline styles with nonce
- **WHEN** Next.js generates inline styles
- **THEN** they include the CSP nonce
- **AND** no CSP violations occur

#### Scenario: inlineCss decision documented
- **WHEN** inlineCss setting is configured
- **THEN** the decision and rationale are documented
- **AND** CSP compatibility is verified

### Requirement: Nonce Generation Single Source
Nonce generation SHALL have a single implementation.

#### Scenario: Single nonce source
- **WHEN** a nonce is needed
- **THEN** it is generated by `src/config/security.ts`
- **AND** other modules re-export, not re-implement

#### Scenario: Consistent validation
- **WHEN** nonce is validated
- **THEN** the same rules apply everywhere
- **AND** length requirements are consistent
